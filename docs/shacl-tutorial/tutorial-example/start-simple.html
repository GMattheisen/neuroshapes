    Start simple

    Let start out with very simple descriptions of the entities in the little publishing domain.

    Researcher description

    A Researcher can be described by a givenName, a familyName as well as its affiliations.

    {
  "@context":"http://schema.org",
  "@type" : "Person",
  "giveName" : "Alice",
  "familyName" : "Alice",
  "affiliations":[]
}
    Property Description
    familyName The researcher family name
    givenName The researcher given name
    affiliations The institutions (labs, universities,…) the researcher is affiliated to
    identifier The researcher persistent digital identifier orcid identifiers
