    Getting Started in 5 mn

    Install the Nexus CLI

    pip install nexus-cli

    Create a new working space in Nexus sandbox

    nexus init

    Set acls

    nexus init

    Define a data model

    {
  "@id" : "{endpoint}/schemas/bbp/core/entity/v1.0.0/",
  "shapes" : [ {
    "@id" : "{endpoint}/schemas/bbp/core/entity/v1.0.0/shapes/Circuit_1_2",
    "@type" : "sh:NodeShape",
    "targetNode" : ["bbp:Circuit_1","bbp:Circuit_2"]
  } ]
}

    Push data

    {
  "@id" : "{endpoint}/schemas/bbp/core/entity/v1.0.0/",
  "shapes" : [ {
    "@id" : "{endpoint}/schemas/bbp/core/entity/v1.0.0/shapes/Circuit_1_2",
    "@type" : "sh:NodeShape",
    "targetNode" : ["bbp:Circuit_1","bbp:Circuit_2"]
  } ]
}
